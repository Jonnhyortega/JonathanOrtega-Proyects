🤖 Jonson-IA 🤖

Bienvenido al servicio de IA de Jonathan Ortega. Usa este endpoint para enviar tus preguntas y recibir respuestas inteligentes basadas en contexto dinámico.

Cómo usar

  1. Envía una petición POST a /api/chat/ desde tu frontend o cliente HTTP.
  2. Incluye en el body un JSON con tres campos:
    • message: tu pregunta o mensaje (string).
    • history: array de intercambios previos [{ role: 'user'|'model', content: '...' }].
    • context (opcional): prompt personalizado para ajustar el asistente.
  3. Recibe en la respuesta el campo response con la respuesta generada.

Ejemplo de petición

{ "message": "Hola, ¿cómo estás?", "history": [], "context": "## Contexto: Eres un asistente amigable y profesional." }

Ejemplo de respuesta

{ "response": "¡Hola! Estoy listo para ayudarte con lo que necesites." }
